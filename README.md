# Sobre el proyecto

**Página web del proyecto:** [https://findrestaurantsdessi.wordpress.com/](https://findrestaurantsdessi.wordpress.com/)

En este proyecto se ha desarrollado una aplicación para la gestión de bares y de restaurantes teniendo en cuenta la situación actual debida al COVID-19. Se divide en dos aplicaciones:
* **Aplicación web**: la aplicación web será usada por los gestores de los bares y restaurantes para organizar sus reservas, atender a sus clientes, ver la disponibilidad de sus mesas en tiempo real, entre otras cosas.
* **Aplicación Android**: la aplicación móvil será usada por los clientes para poder ver todos los restaurantes con la característica de poder ver el **aforo en tiempo real**. De esta manera se intentan evitar las colas innecesarias en la calle y una mayor organización para la situación actual.

El desarrollo de ambas aplicaciones se ha separado en varias prácticas que se detallan a continuación.
